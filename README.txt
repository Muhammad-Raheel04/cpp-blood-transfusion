===============================================
  Blood Transfusion Eligibility Program (C++)
==============================================
Description

This C++ console application determines whether a person is eligible to donate blood. The program performs multiple medical checks such as age, weight, previous donation history, infectious diseases, blood pressure, blood sugar level, Complete Blood Count (CBC), and blood group validation. If all conditions are satisfied, the program also displays which blood groups the donor can donate to.

Features

Age and weight eligibility check
Previous donation waiting period (minimum 8 weeks)
Infectious disease screening (HIV, Hepatitis, Malaria, etc.)
Blood pressure validation
Blood sugar level verification
Complete Blood Count (CBC) analysis
Blood group validation
Blood donation compatibility information

========================================
       Technologies Used
========================================
Language: C++

Concepts: Functions, Function Overloading, Conditional Statements, Loops, Input/Output Handling

========================================
           Program Flow
========================================

User enters personal information (name, age, weight).
Program checks basic eligibility.
Health checks are performed (diseases, blood pressure, sugar level).
CBC values are validated.
Blood group is verified.
If all checks pass, the program displays blood donation compatibility.

========================================
            Outcome
========================================            

The program ensures that only medically eligible donors can donate blood and informs them about compatible recipient blood groups.
